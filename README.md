# Book Trading Club
> A simple book trading website

## Installation

Set up the database:
```sh
mysql &lt ./database/tables.sql -u root -p [Type your server's password]
```

Install/update dependencies:
```sh
npm install
```

Run the webserver:
```sh
node app.js
```

Access the webpage:
```sh
localhost:3000
```

## Built With

* Bootstrap
* jQuery (AJAX Module)
* Node.JS
* Express.JS
* MySQL
